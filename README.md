Install homebrew and formulae:

    ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
    sh Brewfile

Set up vim plugins:

    vim +PlugInstall +qall

Install App Store applications:

* 1Password
* Acorn
* Fantastical
* HTTP Client
* Lingo
* Marked
* Soulver
* The Unarchiver
* Tweetbot
